# CODSOFT Java Development Internship

This repository contains the tasks completed during the CodSoft Java Development Internship.

## Completed Tasks

### Task 1 - Number Game
A console-based number guessing game with:
- Random number generation
- Limited attempts
- Multiple rounds
- Score tracking

### Task 2 - Student Grade Calculator
A Java application that:
- Accepts marks for multiple subjects
- Calculates total marks and percentage
- Assigns grades based on performance
- Validates user input

### Task 3 - ATM Interface
A console-based ATM system using OOP concepts:
- Deposit money
- Withdraw money
- Check account balance
- View account information
- Input validation

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Arrays
- Loops & Conditional Statements
- Input Validation
